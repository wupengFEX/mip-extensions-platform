# mip-youlai-video

mip-youlai-video 实现了三个简单的广告（非必填）+视频的播放器

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-youlai-video/mip-youlai-video.js

## 属性

### ad-src
说明：广告视频的url地址  
必选项：是
类型：string

### ad-src-start
说明：目标视频未播放前，广告视频的url地址  
必选项：是
类型：string

### ad-src-end
说明：目标视频播放完毕之后的广告url地址
必选项：是
类型：string

### target-src
说明：视频的url地址    
必选项：是  
类型：string

### poster
说明：视频的封面图片    
必选项：是  
类型：string
