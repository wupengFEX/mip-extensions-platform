# mip-tanx

阿里妈妈(淘宝联盟)广告组件

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-tanx/mip-tanx.js

## 示例

### 基本用法
```html
<mip-tanx pid='48950672_29684467_110380002'>
    自定义内容，可以嵌套其他组件
</mip-tanx>
```

## 属性

### {pid}

说明：{推广位ID,以"tanx-a-mm_"开头后面的数字}
必选项：{是}
类型：{数字}
